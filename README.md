# Virtual-Assistant-Project 



 

<h2>Description</h2>
The Virtual Assistant project is a project designed to optimize and automate daily tasks, aiming to enhance productivity and improve user experience. This project uses voice recognition and natural language processing technologies, the Virtual Assistant can comprehend and execute a variety of commands, from telling the time and playing music to fetching real-time information.
<br />

- <b>Python 3.6</b> 


<h2>Environments Used </h2>

- <b>MacOs</b>
